# VIdeo-Game-Tracker-Web-Application
Final Java project that uses spring boot and RAWG API to search and display trending games
Game Finder is a web application that allows users to search for video games using the RAWG API, explore game information, and save their favorite titles for quick access later. The application provides an interactive and user-friendly experience for discovering new games and organizing personal favorites.

Features
Search for video games using the RAWG API
View detailed game information including ratings, genres, and release dates
Save favorite games to a personalized favorites list
Remove games from favorites
Responsive and clean user interface
Fast and dynamic search experience
Technologies Used
JavaScript
HTML5
CSS3
RAWG Video Games Database API
Purpose

This project was created to strengthen front-end web development skills, API integration knowledge, and user interface design experience. It demonstrates the ability to work with external APIs, manage application state, and build interactive web applications.

How It Works
Users enter the name of a game into the search bar.
The application sends requests to the RAWG API.
Matching games are displayed dynamically on the page.
Users can add games to their favorites list for later viewing.
